Detailed Description

Custom Functions Unit 0/1 (CFU0/CFU1) HW driver header file.

Author
Stephan Nolting
Warning
There are no "real" CFU driver functions available here, because these functions are defined by the actual hardware.
Hence, the CFU designer has to provide the actual driver functions.
Note
These functions should only be used if the CFU0/CFU1 was synthesized (IO_CFU0_USE/IO_CFU1_USE = true).
